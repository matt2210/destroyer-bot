## Prerequisites

- npm >=5.5.0
- node >=9.3.0

## Install

```sh
npm install
```

## Usage
Step 1 : EDIT CONFIG FILE:
```config.json```
edit this line (3) :
```
"commanddestroy": "COMMAND LAUNCH ATTACK",
```
with your command to destroy like you want for exemple : 
```
"commanddestroy": "!destroyallofmydiscodserver",
```
Step 2 : Save the file ! 
Step 3 : Start the bot  
```sh
node index.js
```
